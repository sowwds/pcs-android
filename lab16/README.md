# Практическая 16: Публикация приложения в RuStore. Ефремов Алексей ЭФБО-10-23

### 1. Информация о проекте

-   **Название приложения:** Habit Tracker (Трекер Привычек)
-   **Цель:** Предоставить пользователям простой, минималистичный и мотивирующий инструмент для формирования и отслеживания ежедневных привычек.
-   **Технологии:** Flutter, Supabase (аутентификация, база данных).

### 2. Подготовка к публикации (Этап 1)

Перед публикацией проект был полностью подготовлен в соответствии с требованиями предыдущих практических работ:

1.  **Исправлены ошибки (Пр 14):** Проведен статический анализ с помощью `flutter analyze`, все найденные проблемы были устранены.
2.  **Написаны тесты (Пр 14):** Были добавлены unit- и widget-тесты для проверки ключевой логики и компонентов UI.
3.  **Оптимизирован код (Пр 15):** Применены базовые оптимизации, включая использование `const` и `ValueKey` для повышения производительности.
4.  **Добавлены иконка и splash-экран (Пр 15):** С помощью пакетов `flutter_launcher_icons` и `flutter_native_splash` были сгенерированы иконки приложения и экран загрузки.
5.  **Настроена подпись (Пр 15):** Была настроена конфигурация в `build.gradle.kts` для подписи релизных сборок с использованием ключа разработчика.

После выполнения всех шагов была успешно собрана релизная версия приложения в формате App Bundle (AAB) с помощью команды:
```bash
flutter build appbundle --release
```

### 3. Регистрация в RuStore и создание карточки приложения (Этапы 2-3)

На платформе `console.rustore.ru` была выполнена регистрация аккаунта разработчика и создана карточка для нового приложения. Были заполнены все необходимые поля:
-   Название приложения: **Habit Tracker**
-   Категория: **Здоровье**
-   Краткое и полное описания.
-   Возрастное ограничение: **0+**.

Также были загружены необходимые материалы: иконка приложения (512x512) и файл политики конфиденциальности (`privacy_policy.txt`), созданный на предыдущем этапе.

### 4. Загрузка сборки и модерация (Этапы 4-5)

Во вкладке "Версии" была добавлена новая версия приложения:
-   **Версия:** 1.0.0
-   **Код сборки:** 1

Собранный файл `app-release.aab` был успешно загружен в систему. После заполнения всех данных приложение было отправлено на модерацию. На текущий момент оно находится в статусе "в обработке".
